HACKATHON PROJECT

I took part in my third hackathon, the Code Institute's "Women In Tech" Hackathon in March 2025. Our team created Connection Engine, a platform designed to empower and support women in tech through visibility, mentorship, and access to real-world open-source projects. The site features dedicated sections for women developers, open-source maintainers, mentors, and sponsors — all working together in a shared space to create real opportunities.

Users can explore and apply for fully funded open-source projects, sign up to become a mentor or find one, and browse available sponsorship opportunities. Maintainers can list their projects and match them with mentors or developers. We also integrated a secure Stripe payment system for donations and sponsorships, complete with a success page and project visibility boosts.

The platform follows a retro design theme inspired by the 80s and 90s, with pixelated UI, custom illustrations, and vibrant color schemes. All users can register under one of three roles: contributor, mentor, or maintainer and get access to unique tools and dashboards depending on their role. Interactive UI elements, light/dark mode, and responsive design were key focuses in the UX.

I worked mainly on the backend of the project, where I implemented custom user types for contributors, mentors, and maintainers, as well as built the contact page functionality. We had a fantastic team where everyone collaborated beautifully, supported each other, and shared a clear vision. We are considering further developing the app after the hackathon. I’m incredibly proud to say that we won the hackathon! 🏆

Juridiq is a legal web application designed to help users create and manage legal cases, request legal advice, submit appeals, and access legal documents. The platform was built from scratch using React for the frontend and Django REST Framework for the backend. It provides a user-friendly interface with clearly structured flows for both private users and legal professionals.

The application includes features such as two-factor authentication (2FA), Stripe payments, document uploads, status tracking for cases, and secure account management. Future updates will introduce more advanced functionalities, including document templates and automatic legal form generation. NEON Tech is used as the production database, while SQLite3 is used in development.
